Detailed Description

The MainWindow class intialises the main window and its components for the application and for it to work with VTK In the MainWindow class, it connects the components from the ModelPartList and the vtk renderer to the MainWindow for the user to be able to interact with the ui. The class allows for rendering and holding of values, the user to perform file actions and button interactions on the ui.
